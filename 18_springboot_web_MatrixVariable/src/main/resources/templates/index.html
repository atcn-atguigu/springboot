1、测试@MatrixVariable - 用法一（数组类型矩阵变量在URL中分开写）

@PathVariable("role") String role, // ⚠️有@PathVariable, @MatrixVariable才能被解析使用
@MatrixVariable("username") String username,
@MatrixVariable("interests") List<String> interests,
@MatrixVariable("age") Integer age
"/testMatrixVariable/manager;username=Ben;interests=basketball;interests=badminton;age=22" --> success.html
"/testMatrixVariable/developer;username=Tom;interests=baseball;interests=football;age=28" --> success.html

1、测试@MatrixVariable - 用法一（数组类型矩阵变量在URL中合并写）

"/testMatrixVariable/manager;username=Ben;interests=basketball,badminton;age=22" --> success.html
"/testMatrixVariable/developer;username=Tom;interests=baseball,football;age=28" --> success.html

2、测试@MatrixVariable - 用法二

@MatrixVariable(value = "age", pathVar = "bossId") Integer bossAge,
@MatrixVariable(value = "age", pathVar = "empId") Integer empAge
"/testMatrixVariable/1;age=20/2;age=10" --> success.html
